# 🚢 Task 5: Exploratory Data Analysis (EDA) - Titanic Dataset

## 📌 Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to discover meaningful patterns, relationships, trends, and insights using statistical analysis and data visualization.

---

## 📂 Dataset

- Dataset: Titanic Dataset (Kaggle)
- Records: 891
- Features: 12

---

## 🛠 Tools & Libraries

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow

### 1. Data Loading

- Imported dataset
- Previewed dataset

### 2. Data Understanding

- Dataset Information
- Shape
- Columns
- Statistical Summary
- Missing Values
- Duplicate Records

### 3. Data Cleaning

- Removed Cabin column
- Filled missing Age values using Median
- Filled missing Embarked values using Mode
- Verified cleaned dataset

### 4. Exploratory Data Analysis

#### Univariate Analysis

- Survival Count
- Gender Distribution
- Passenger Class Distribution
- Age Distribution
- Age Box Plot

#### Bivariate Analysis

- Survival vs Gender
- Survival vs Passenger Class
- Survival vs Age
- Survival vs Fare

#### Multivariate Analysis

- Correlation Heatmap
- Pair Plot

---

## 📈 Key Insights

- Most passengers did not survive.
- The majority of passengers were male.
- Most passengers traveled in Third Class.
- Female passengers had a significantly higher survival rate than male passengers.
- First Class passengers had a much higher survival rate than Third Class passengers.
- Passengers who paid higher fares generally had a better survival rate.
- Most passengers were between 20 and 40 years old.
- Fare and Passenger Class showed a noticeable correlation.

---

## 📌 Conclusion

Exploratory Data Analysis revealed that passenger class, gender, and ticket fare had a significant influence on survival. Data cleaning improved dataset quality, while visualizations helped uncover important relationships and trends.
